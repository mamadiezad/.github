# .github 🏛️

**Community Health Files** — default issue & PR templates for all repositories under [@mamadiezad](https://github.com/mamadiezad).

These templates are automatically applied to every repository that doesn't have its own templates.

## 📋 Included Templates

| Template | Description |
|----------|-------------|
| 🐛 `bug_report.md` | Structured bug reports with environment info |
| 🚀 `feature_request.md` | Feature suggestions with implementation ideas |
| 📦 `PULL_REQUEST_TEMPLATE.md` | PR checklist with testing & changelog |
