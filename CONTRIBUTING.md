# 🤝 مشارکت / Contributing

از مشارکت شما در این پروژه خوشحالیم!

## نحوه مشارکت

1. **فورک** (Fork) کنید
2. یک **برنچ** جدید بسازید (`git checkout -b feature/YourFeature`)
3. تغییرات خود را **کامیت** کنید (`git commit -m '✨ Add YourFeature'`)
4. **پوش** کنید (`git push origin feature/YourFeature`)
5. یک **Pull Request** باز کنید

## استانداردهای کد

- از TypeScript strict mode استفاده کنید
- کد را با `npm run lint` چک کنید
- تست‌ها را پاس کنید
- از Conventional Commits استفاده کنید

## گزارش باگ

از قالب **Bug Report** در Issues استفاده کنید.

## درخواست ویژگی

از قالب **Feature Request** در Issues استفاده کنید.
